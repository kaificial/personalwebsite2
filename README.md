# personal portfolio site
my personal website built with next.js to showcase my experience and work to hopefully land a job 

## running it locally

```bash
cd website
npm install
npm run dev
```

then open [localhost:3000](http://localhost:3000)

## built with

- next.js
- react
- framer motion
- scss

## features

- dark/light mode
- responsive design for mobile
- animated ui
- projects
- experience timeline
